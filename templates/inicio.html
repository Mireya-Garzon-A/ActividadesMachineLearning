Aplicación de Machine Learning Supervisado

ML supervisado es una técnica que entrena modelos a partir de datos
etiquetados para predecir o clasificar resultados futuros con base en patrones aprendidos.

Regresión Lineal

La regresión lineal en Machine Learning es un algoritmo supervisado que modela la relación entre una variable independiente y una dependiente mediante una línea recta para predecir valores continuos..

Regresión Logística

La regresión logística es un algoritmo de Machine Learning usado para clasificar datos en categorías, estimando probabilidades entre 0 y 1 mediante una función sigmoidea. Es sencillo, rápido y ampliamente aplicado en tareas como detección de fraudes, diagnóstico médico y análisis de abandono de clientes..
